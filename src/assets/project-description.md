# 项目概览
做一个有创意的可交互的大海场景的个人主页

# 网页设计
1. 打开就是类似于 `assets/shaders/ocean-water-test`的场景
2. 海水要是清透的tiffany蓝
3. 沙滩是粉色沙滩，类似于科摩多岛的粉色沙滩，有一些可爱的贝壳和海星
4. 镜头一开始俯瞰沙滩和海岸，沙滩上出现 “Hi, I am Mengyu Han”
5. 海浪冲刷这行字，海滩上出现中文的“你好，我是韩梦宇”
6. 海里再次冲刷，用户可以开始自由移动视野。用户视野一开始在沙滩上，抬头可以看到大海和海浪
7. 大海里有一个类似于 `assets/shaders/artifact-at-sea`中的神秘有漂亮的，会发光的立方体
8. 这个立方体怎么交互我还没想好，先变成一个神秘的东西，之后可能变成作品集的入口吧
9. 用户长按沙滩的时候就可以在沙滩上写字画画
10. 未来有空在沙滩的某个区域可以留个菜单或者神秘入口，隐藏一些东西在里面
